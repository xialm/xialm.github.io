博客迁移 Github Octopress

之前使用的博客一直是wordpress，前些日子因为忘记给空间续费导致写的几篇博文丢失。联系了空间服务商被告知只要欠费即会删除，且无备份。
非常生气，好一些的空间每个月几十块对我这个刚毕业的学生来说也是不小的负担。另外加上伪极客气质，所以尝试Github+Octopress的组合。虽说有被墙的风险，而且访问慢到没朋友，不过即免费又相对稳定也就能达到我的预期啦。
相对于傻瓜式的wordpress，octopress的确是稍微复杂一些，经常用github的盆友轻车熟路。对于我这半只脚还没入门的菜鸟来说折腾一下也能很快的搭建好。
新的开始，所见所闻所知将会尽可能的整理到这里，也希望自己可以坚持下来。
